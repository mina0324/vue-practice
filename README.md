### 项目自动启动
+ yarn install  安装依赖
+ 然后通过 CMD 运行 start.bat 即可


### 技术栈

+ Vue2 + VueRouter + Vuex + Axios + Less + Json-Server + NodeJs + express(只做了JWT生成与校验功能的后台,其他的功能都是用的JSONServer)
